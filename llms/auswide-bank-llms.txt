# Auswide Bank

> Auswide Bank Ltd is an Australian authorised deposit-taking institution (ADI) and an active Consumer Data Right (CDR) data holder. It exposes a public, unauthenticated Product Reference Data (PRD) API conforming to the DSB Consumer Data Standards, alongside the accredited-data-recipient consumer-data-sharing channels required of every Australian bank. Auswide is now a division of MyState Bank Limited (ASX: MYS) following the 2025 merger.

## APIs

- [Auswide Bank CDR Product Reference Data API](https://www.auswidebank.com.au/help/banking-support/open-banking/): Public, unauthenticated CDR Product Reference Data (PRD) API exposing Auswide's publicly available banking products via GET /banking/products and GET /banking/products/{productId}. Live base: https://api.auswidebank.com.au/openbanking/cds-au/v1 (HTTP 200, x-v 4, 28 products).
